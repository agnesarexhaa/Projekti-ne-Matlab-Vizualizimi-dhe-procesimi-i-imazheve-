# Projekti-ne-Matlab-Vizualizimi-dhe-procesimi-i-imazheve-
Ky projekt paraqet aplikimin e teknikave të përpunimit të imazheve përmes MATLAB-it. Qëllimi kryesor është të demonstrohet përdorimi i funksioneve të ndryshme për manipulimin, analizimin dhe përmirësimin e imazheve. Gjithashtu, përfshihet segmentimi i imazhit dhe përdorimi i filtrave për përmirësimin e cilësisë.
